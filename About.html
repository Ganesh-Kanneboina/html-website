HTML Basics

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. "HyperText" means text that has links to other pages (like clicking a link to go to another website). "Markup Language" means it uses tags to describe the structure of a web page (like headings, paragraphs, images, videos, etc.).

Why do we use HTML?
  1. To Create Web Pages
    • HTML allows us to build the structure of a web page (like a skeleton).
  2. To Display Content
    • It helps display text, images, videos, links, buttons, and more on a website
  3. To Organize Information
    • HTML uses tags to divide content into headings, paragraphs, sections, etc.
  4. To Link Between Pages
    • HTML allows us to link one page to another using the (anchor) tag.
  5. To Embed Media
    • We can embed videos, audio, images, and maps in web pages using HTML.
  6. To Work With CSS and JavaScript
    • HTML works together with:
    • CSS to style the page (colors, layout, fonts)
    • JavaScript to add behavior (like clicking a button)
